# `trello search`

Search Trello boards and cards.

## Subcommands

- `trello search cards --query <text>`
- `trello search boards --query <text>`

## Examples

```bash
trello search cards --query "docs"
trello search boards --query "marketing"
```

## Usage Pattern

Use search when you need a text-first discovery step and do not already know the board or card IDs you need.
